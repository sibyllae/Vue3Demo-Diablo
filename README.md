# D2R 攻略網 — 暗黑破壞神 II：獄火重生

> Diablo II: Resurrected 符文與符文之語完整攻略資料庫

## ✨ 特色

- 🎮 **完整符文資料** — 33 顆符文（El → Zod）的等級、武器/防具效果
- ⚔️ **經典符文之語** — 12+ 個最熱門符文之語的完整屬性
- 🔍 **即時搜尋** — 依名稱、效果、屬性快速篩選
- 🌑 **暗黑風格 UI** — Diablo 主題的暗色金紅配色設計
- 📱 **響應式設計** — 手機、平板、桌面完美呈現

## 🛠️ 技術棧

- **Vue 3** — Composition API + `<script setup>`
- **Vite 6** — 極速開發體驗
- **Vanilla CSS** — 自訂設計系統，無框架依賴

## 🚀 本地開發

```bash
npm install
npm run dev
```

## 📦 建置部署

```bash
npm run build
npm run preview   # 本地預覽建置結果
```

## 🌐 GitHub Pages 部署

Push 到 `main` 分支即自動部署。

手動部署：
1. 進入 GitHub repo → Settings → Pages
2. Source 選 **GitHub Actions**
3. Push 程式碼即可
